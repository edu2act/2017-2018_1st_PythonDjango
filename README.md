# Python Django
>（2017~2018 第一学期）
Python Django 是一门基于Python的Django框架的课程，课程包括俩部分，第一部分简单介绍Python的基础知识，第二部分详细介绍Django框架的应用。全课程以编写一个图书馆管理系统为例，分别介绍了Djang各个部分的应用。
